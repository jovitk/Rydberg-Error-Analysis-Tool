
# Rydberg & Error Analysis Tool

A simple GUI application for **CHY1005: Introduction to Computational Chemistry** that combines Rydberg calculations and error propagation in one place.

---

##  Features

- Calculate wavelength using Rydberg equation
- Identify spectral series (Lyman, Balmer, etc.)
- Basic spectrum visualization
- Error calculation for addition, subtraction, multiplication, and division
- Shows result with uncertainty and relative error

---

##  Tech Stack

- Python
- CustomTkinter
- NumPy
- Matplotlib

---

##  How to Run

```bash
git clone https://github.com/jovitk/Rydberg-Error-Analysis-Tool.git
cd Rydberg-Error-Analysis-Tool
pip install customtkinter matplotlib numpy
python project3.py
```
